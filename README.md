# Developer. Athlete. Gamer. Portfolio

A personal portfolio built with Next.js (App Router), Tailwind CSS, and Framer Motion.

## Vibe
Clean, modern, high-performance dark-themed aesthetic. Refines the "Developer, Athlete, Gamer" identity.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS (v4)
- **Animations:** Framer Motion
- **Language:** TypeScript

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure
- `src/app/page.tsx`: Home Page with Hero section and Status Badge.
- `src/app/layout.tsx`: Root Layout including the Navbar.
- `src/components/Navbar.tsx`: Responsive navigation component.
